DOM - Document Object Model

This section of the code is intended to demonstrate the usage of the `getElementById` method in JavaScript. The `getElementById` method is used to access an HTML element based on its unique `id` attribute. It returns a reference to the element, allowing you to manipulate its properties, attributes, and content.

Using getElementsByClassName

This section of the code is intended to demonstrate the usage of the `getElementsByClassName` method in JavaScript. The `getElementsByClassName` method is used to access all HTML elements that have a specific class name. It returns a live HTMLCollection of elements, which can be accessed and manipulated using JavaScript. Unlike `getElementById`, which returns a single element, `getElementsByClassName` can return multiple elements.
