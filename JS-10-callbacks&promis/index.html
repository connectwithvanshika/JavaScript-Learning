Callbacks and Promises in JS/Asynchronous behaviour of JS

Callbacks: Functions passed as arguments to other functions, executed once the task is complete.

Promises: Objects that represent the eventual completion (or failure) of an asynchronous operation.

Synchronous: Code executed in sequence, blocking the next task until the current one finishes.Solution to callback hell is promises

Asynchronous: Code that allows the next task to run while waiting for another task to finish (non-blocking).
